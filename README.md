# LanCheck
это простая утилита для сканирования открытых портов на локальном компьютере (127.0.0.1). Она помогает быстро выявить потенциально опасные открытые порты и принять меры. Программа полностью безопасна, не требует прав администратора и доступна на двух языках.

Интерфейс интуитивно понятен: запустите сканирование, дождитесь результатов, прочитайте рекомендации. Открытые порты 135 и 445 являются штатными для Windows, остальные — повод задуматься о настройке брандмауэра.


This is a simple utility for scanning open ports on your local computer (127.0.0.1). It helps you quickly identify potentially dangerous open ports and take action. The program is completely safe, does not require administrator rights, and is available in two languages.

The interface is intuitive: run the scan, wait for the results, read the recommendations. Open ports 135 and 445 are standard for Windows; others are a reason to think about configuring your firewall.
